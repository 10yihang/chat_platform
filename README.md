# 在线聊天网站

这是一个在线聊天网站，允许用户实时聊天、上传和共享文件、播放音乐和视频，以及进行语音和视频通话。

## 功能

- 实时聊天功能
- 文件上传和共享
- 音频和视频播放
- 语音通话和视频通话

## 技术栈

- 后端：Python, Flask
- 前端：React, TypeScript

## 项目结构

```
chat-website

│   └── README.md
└── frontend

```

## 安装和运行

1. 克隆项目：
   ```
   git clone <repository-url>
   ```

2. 进入后端目录并安装依赖：
   ```
   cd backend
   pip install -r requirements.txt
   ```

3. 运行后端：
   ```
   python src/app.py
   ```

4. 进入前端目录并安装依赖：
   ```
   cd frontend
   npm install
   ```

5. 运行前端：
   ```
   npm start
   ```
